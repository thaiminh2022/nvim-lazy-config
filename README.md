# Thaiminh2022's Neovim config

## Install the config
1. Make sure to remove or backup your current `nvim` directory
2. Clone this repo to your config

```
# Windows
git clone https://github.com/thaiminh2022/nvim-lazy-config.git ~/AppData/nvim
```
3. Run `nvim` and wait for the plugins to be installed

## This repo contains some core plugins

- lsp-zero for easy lsp acess
- telescope
- undotree (now built in)
- cool colorscheme
- Flash for jumping around
- lualine + whichkey for visual
